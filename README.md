# FuturiMarket

A futuristic online marketplace for buying and selling websites, SaaS products, apps, and digital businesses.

## Features

- Live digital marketplace UI with glassmorphism, neon accents, and animated interactions
- Buyer/seller marketplace concept with featured listings, revenue, traffic, and growth potential
- Real-time simulated updates for revenue and market activity
- Dark-mode design with premium tech visuals

## Run locally

1. `cd C:\Users\Ender\Projects\FuturiMarket`
2. `npm install`
3. `npm run dev`

Open `http://localhost:3000`
